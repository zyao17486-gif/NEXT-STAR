# Next Star — 商业化思考

> 面试必答题：「这个产品怎么赚钱？」「市场规模多大？」

## 用户分层

| 层级 | 画像 | 规模估算 | 付费意愿 |
|------|------|---------|---------|
| 泛球迷 | 看总决赛但不关注选秀，需要一个「入坑路径」 | ~500 万（中国 NBA 观众 × 20%） | 低 |
| 进阶球迷 | 关注选秀但没有固定信息来源，虎扑水帖为主 | ~50 万 | 中 |
| 硬核球迷 | 已看 ESPN/Tankathon，需要中文效率工具 | ~10 万 | 高 |
| 篮球从业者 | 自媒体/训练师/经纪人，需要结构化球员数据 | ~2 万 | 极高 |

## 变现路径（按时间线）

### Phase 1：免费增长期（现在）
- **目标**：积累用户基数和内容资产
- **策略**：核心功能全免费，用「关注+AI 推荐」建立留存
- **KPI**：月活 > 5,000，关注转化率 > 30%

### Phase 2：Freemium 分层（月活 > 10K 后）

| 功能 | Free | Premium（¥19/月） |
|------|------|-------------------|
| 球员浏览 | ✅ 前 20 名 | ✅ 全部 |
| AI 推荐 | ✅ 1 次/周 | ✅ 无限 |
| 球探台搜索 | ❌ | ✅ |
| 球员对比 | ❌ | ✅ 1v1 |
| 上下限模板 | ✅ 基础 | ✅ 深度分析 |
| 选秀倒计时 | ✅ | ✅ 股票式追踪 |
| 数据导出 | ❌ | ✅ CSV/API |

### Phase 3：B 端变现

| 产品 | 目标客户 | 定价 |
|------|---------|------|
| 球员数据库 API | 篮球自媒体/数据分析师 | ¥99-499/月 |
| 嵌入式推荐 Widget | 篮球社区/App | 收入分成 |
| 球探报告白标 | 训练营/青训机构 | 按份定价 |

## 市场规模估算（中国市场）

```
中国 NBA 观众 ≈ 2,500 万人
其中关注选秀内容的人群 ≈ 200 万人（8%）

TAM（总可寻址市场）：200 万 × ¥19/月 × 12 月 = ¥4.56 亿/年
SAM（可服务市场，付费意愿 > 30%）：≈ ¥1.4 亿/年
SOM（首年可获取，保守估计 0.5%）：≈ ¥228 万/年
```

> 注：以上为自上而下的粗略估算，实际验证需通过 MVP 阶段的转化漏斗数据迭代。

## 增长飞轮

```
AI 翻译生成内容（零边际成本）
    ↓
球员数据库规模扩大（结构化成壁垒）
    ↓
更多中文用户通过搜索进入（SEO 长尾）
    ↓
用户关注行为产生偏好数据
    ↓
推荐算法更精准（数据网络效应）
    ↓
付费转化率提升 → 再投入内容生产
```
